# Screen Veil store kit

This folder contains:
- `screenveil-extension-source/` — editable extension source
- `screenveil-extension-store-ready.zip` — upload this to the Chrome Web Store
- `store-listing-copy.md` — text for the listing and privacy answers
- `landing-page/` — homepage files
- `privacy/` — privacy policy page
- `support/` — support page
- `install/` — stable install URL page to redirect later to the real store listing
- `assets/` — icon, promo, marquee, screenshot
- `publish-checklist-screenveil.md` — practical launch checklist
- `screenveil-billing-login-plan.md` — Lemon Squeezy + account-login rollout plan
