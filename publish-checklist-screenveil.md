# Screen Veil publish checklist

## Domain and website
- [ ] Buy `screenveil.app`
- [ ] Point DNS to your hosting provider
- [ ] Publish homepage at `https://screenveil.app`
- [ ] Publish privacy policy at `https://screenveil.app/privacy`
- [ ] Publish support page at `https://screenveil.app/support`
- [ ] Publish install page at `https://screenveil.app/install`

## Extension package
- [ ] Upload `screenveil-extension-store-ready.zip`
- [ ] Confirm the extension name shows as `Screen Veil`
- [ ] Confirm icon displays correctly
- [ ] Confirm built-in presets, custom domain protection, language selection, and blur strength still work

## Store listing
- [ ] Paste copy from `store-listing-copy.md`
- [ ] Upload `assets/screenveil-icon-128.png`
- [ ] Upload `assets/screenveil-promo-440x280.png`
- [ ] Upload at least 1 screenshot; ideally 3 to 5
- [ ] Add homepage, privacy, and support URLs
- [ ] Review permissions explanation carefully

## After first upload
- [ ] Copy the real Chrome Web Store listing URL
- [ ] Replace `REAL_CHROME_WEB_STORE_URL` inside `install/index.html`
- [ ] Re-deploy the install page

## Billing later
- [ ] Create Screen Veil website account system
- [ ] Create Lemon Squeezy monthly and yearly products
- [ ] Implement checkout endpoint
- [ ] Implement Lemon Squeezy webhook endpoint
- [ ] Add extension entitlement check after login
